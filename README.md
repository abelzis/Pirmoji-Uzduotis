# Pirmoji Užduotis
OOP pirmoji užduotis. Programa įrėmina įvestą vardą ir su juo pasisveikina.

# Kaip paleisti programą?
Lengviausias būdas - tai parsisiųsti First_Task.exe failą ir jį paleisti.
